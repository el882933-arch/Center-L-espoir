## 📖 About the Project

This project is the **official website of Al-Amal Center**, a real-world educational institution. The website aims to introduce the center, highlight its educational mission, and provide comprehensive information about the courses and services it offers — while making it easy for visitors to access essential information in a professional and user-friendly way.

This project was developed by **BitBrief** as a token of appreciation for a teacher who had a positive impact on our educational journey, believing that the best way to give back is by building tools that serve the community and help spread knowledge.

> **Note:** This website is not an e-learning platform or a place for delivering online lessons. Rather, it serves as the official digital face of the center, aiming to strengthen its online presence and provide a great experience for students, parents, and visitors.

## 📥 Installation

**1. Clone the repository:**

```bash
git clone https://github.com/el882933-arch/Center-L-espoir.git
```

**2. Navigate into the project folder:**

```bash
cd Center-L-espoir
```

**3. Install dependencies:**

```bash
npm install
```

---

## ▶️ Running the Project

To run the website in development mode:

```bash
npm run dev
```

To run it with access over the local network:

```bash
npm run dev -- --host 0.0.0.0
```

---

## 🏗️ Building for Production

```bash
npm run build
```

You can then preview the final build with:

```bash
npm run preview
```

## ⚠️ Common Issues

<details>
<summary><strong>Error during package installation</strong></summary>

If you encounter an error related to `npm.mirrors.msh.team`, delete the following files:

```bash
rm -rf node_modules
rm -f package-lock.json
```

Then reinstall:

```bash
npm install
```

</details>
